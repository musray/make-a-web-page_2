What is web development

Web development, is the activities of developing a web site for internet. It covers very broad works such as front-end, back-end, web page design and so forth.

Web development can range from making the single web page of plain text to the most complex web applications, e-comercial and SNS, etc.

How internet works

"The internet is the network of networks".

When a people type a URL in his/her web browser and hit ENTER, the browser, also know as the client, sends a HTTP request to the internet. Then a machine called router, which is connecting to the people's computer, take this request and try to find the best or the most optimal path to the destination where this request is sending to.

There might be a few of routers this request will hopping through before it will arrived at the SERVER at the other end of the internet. The server, might be a computer running Unix or Linux system, will interpret this request and find the related HTML document to this request. Then the server will sent the documents back to the internet, again, through many routers and get the people's computer and browser. The browser interprets the HTML file and renders it according to the structure and content of the HTML, and its companied CSS rules, to a particular web page.

Programmers are lazy

It's true, programmers are lazy. Because they don't like any repetition in their work, they always try hard to find some way to avoid the repetition.

Repetition has some disadvantages:

  • Waste time
  • Introduce errors
  • Hard to review

Since programmers don't like repetition, how to avoid it?

  • Put the always desired codings inside a piece of source code.
  • Name the piece of source code. We call the piece of code a function, or a procedure.
  • Directly call this function by name other than write all the code time by time whenever it's needed.
