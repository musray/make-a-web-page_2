What is HTML?

HTML is the term Hyper Text Markup Language for short. As an markup language, HTML describes the structure of the web page.

Since structure is important for reader to understand, navigate an web page, it is very important to understand how HTML works.

How HTML describes structure of a web page?

HTML uses tag to make element. Elements bracketed by the tags are like some boxes. Different shape of boxes in different position consists the whole structure of web page.

The elements are usually made up of an opening tag and a close tag(has an back slash in it).Some tags which only have a open tag but without an closing tag are empty element.
